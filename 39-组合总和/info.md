# 39

* 问题

  给**一组**候选数字**（C）（没有重复）**并给一个目标数字**（T）**，找出 **\*C ***中所有唯一的组合使得它们的和为 **\*T***。

  可以从 **C **无限次数中选择**相同**的数字。

  **说明：**

  - 所有数字（包括目标）都是正整数。
  - 解集合中没有相同组合。

   

  例如，给一个候选集合 `[2, 3, 6, 7]` 并且目标为 `7`，
  一个解的集合为：

  ```
  [
    [7],
    [2, 2, 3]
  ]
  ```

* 解决

  * 递归的思路

    排序之后，在每次递归中从最小的元素开始尝试，选择目标减去当前值后向下递归，需要注意提前判断结束以提高效率。还有注意在s=0时候也就是找到了一个解，加入到结果的列表中。
